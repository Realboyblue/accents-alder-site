# Accents & Alder site maintenance

## Update services
Edit:
- content/services.json

## Update gallery
1) Add images to:
- assets/img/gallery/

2) Add entries to:
- content/gallery.json

Example item:
{
  "title": "Wainscoting stair wall",
  "desc": "Clean panel spacing with refined finish.",
  "category": "wainscoting",
  "image": "assets/img/gallery/wainscot_01.jpg"
}

## Publish
Commit + push to GitHub. Netlify deploys automatically.
